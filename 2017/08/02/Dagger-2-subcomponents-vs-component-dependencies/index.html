dagger2 : 组件依赖(Component)和子组件(Subcomponent)的区别

example 依赖

  • SomeClassB1依赖SomeClassA1。provideSomeClassB1 方法显示了它们之间的依赖关系。

example 组件依赖

  • SomeClassB1依赖SomeClassA1。 ComponentB需要显式声明依赖， ComponentA不需要声明ModuleB。
  • 组件依赖保持两个组件独立性。

example 子依赖

  • SomeClassB1依赖SomeClassA1。 ComponentB不需要显式声明依赖。ComponentA需要声明ModuleB。
  • 子依赖使得两个组件耦合。
